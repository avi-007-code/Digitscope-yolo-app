# 🔍 DigitScope: Smart Number Detector

DigitScope is an intelligent web app built with **Streamlit** and **YOLO** that detects numbers in uploaded images using advanced AI vision.  
Switch between Dark and Light themes, upload any image, and let DigitScope instantly reveal all the digits! ⚡🔢

---

## 🌐 Live Demo
[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://digit-scope.streamlit.app)

---

## 🖼️ Features

✅ **Dark & Light Mode** theme toggle  
✅ **YOLOv4/YOLOv11n** model integration for number detection  
✅ Clean UI built with **Streamlit + Custom CSS**  
✅ Upload and preview any `.jpg`, `.jpeg`, or `.png` image  
✅ Instant AI detection and visual bounding boxes  
✅ Works on both **Desktop & Mobile**
